Reports problems that prevent rendering an API Blueprint (`.apib`) preview using the external `aglio` renderer.

API Blueprint Viewer relies on the `aglio` command-line tool to convert API Blueprint source into HTML for preview. If `aglio` is not installed, not found on your system, or not configured in the plugin settings, rendering cannot run and this inspection will highlight the issue.

Typical fixes:

  • Install `aglio` (requires Node.js): 
    npm install -g aglio
  • Configure the `aglio` toolchain location in Settings | Tools | API Blueprint.

Notes:

  • `aglio` is an external tool and must be available on your machine. The plugin may try to discover it from common locations (e.g., system `PATH`), but you can always set the toolchain path explicitly.
  • If you change your Node.js / npm global install location, you may need to reconfigure the toolchain path.
